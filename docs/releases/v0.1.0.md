# Code::Blocks Stable Toolchain Edition v0.1.0

## Release summary

This first public release mirrors the **known-good local Code::Blocks setup** that was already working correctly, including the debugger configuration, STL pretty-printers, and managed watch-window behavior needed for useful C++ debugging.

## Included baseline

- Code::Blocks `25.03`
- Bundled MinGW-w64 GCC/G++ `14.2.0`
- Bundled GDB `16.2`
- Managed profile seed captured from the known-good local baseline
- Machine-wide installer target: `C:\Program Files\CodeBlocks Stable Toolchain Edition`

## What makes this release plug-and-play

- compiler, linker, and debugger are pinned by absolute path
- the managed profile seed preserves the working pretty-printer init chain
- `libstdc++` printers are registered for better STL watch rendering
- the dev-only `C:\Devel\CodeBlocks\...` debugger source line is stripped from the staged payload
- backup-first migration protects the existing roaming profile before seeding the managed one

## Installer notes

- architecture: Windows 10/11 x64 hosts
- target builds: Win32 and x64
- installer signing: **unsigned**
- PATH modification: disabled by default
